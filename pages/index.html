MoGIT

协助设计师同步项目图片的简化版 CODE 客户端。

1.安装本地环境

  1. 下载并安装 Git for Mac 本地下载 | 官网 (这货装完后什么都不会发生)。
  2. 下载 MoGIT，按照提示安装好后看第二步。

2.打开MoGIT

  1. 按照提示输入你的内网帐号密码，点击设定。忘记了？
  2. 在图示位置输入工程师给你的Git地址，比如「http://code.dapps.douban.com/bear/mogit.git」，点击设定。
  3. 恭喜你配置完成！快去桌面的Mogit目录里，用你喜欢的方式添加文件吧！
  4. 本地文件夹的图片添加完毕，在红框位置输入本次提交的理由（必填)，点击「同步」之后就可以提交到项目里了。

Tips

  1. 如需更换项目，点击「锁」。
  2. 如需更新本地内容，点击logo「袋鼠」。
  3. 使用过程中如果遇到了任何诡异的问题，请联系@bear或@whshang。

Q & A

  1. 为什么不是直接通过skype传文件给工程师？

    除非你和工程师都是单枪匹马，或者共享的文件用两只手能数的清楚，一旦多人协作或涉及到大量文件更新，从文件命名的规范到部分文件更新，用IM直传都不是一个好的解决方案。

  2. 为什么是Git而不是dropbox？

    • 内网提交速度更快
    • 多人共同修改同一份文件时不会被覆盖，随时可以恢复到一个以前版本
    • 云端存储，可以很方便的在线查看和检索
    • 与工程师协作更直接

  3. 本工具适用的项目类型

    • 移动（尤其是Android）项目，需要大量素材文件
    • 设计师需要手动修改html/css

One More Thing

MoGIT只是协同开发第一步，完整体验GIT需要命令行或其他更强大的GIT工具。
