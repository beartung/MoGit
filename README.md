#p.s. 如果你内心足够强大，也可以自己试着用用看：

  1. 设置LDAP（内网）帐号
  2. 填项目code地址
  3. 在对应文件夹里用你最熟悉的方式折腾
  4. 回到Mogit，填提交信息，点同步按钮
  5. 有任何诡异的情况，联系@bear或@whshang
